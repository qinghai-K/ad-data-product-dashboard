# 广告投放数据分析与数据产品看板项目
本项目是一个面向数据产品实习岗位的练习项目，模拟广告投放业务场景，完成广告核心指标理解、SQL 数据分析、BI 看板设计、badcase 整理和数据产品文档输出。

## 项目目标

- 理解广告投放核心指标
- 使用 SQL 分析广告数据
- 使用 Excel / Power BI 搭建广告效果看板
- 设计 badcase 规则并整理异常案例
- 输出指标口径文档和看板需求文档

## Day 1 学习内容

- 理解广告投放基础链路
- 学习曝光、点击、转化、消耗等基础字段
- 学习 CTR、CVR、CPC、CPA、CPM、ROI 等核心指标
- 初步理解 badcase 和 BI 看板的作用
